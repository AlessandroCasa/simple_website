Include Images

Including an image using rmarkdown is easy.

In the files pane in RStudio choose New Folder (or create one in your Finder in the Rproject folder). Call it say images.

To reference an image in rmarkdown use the following.

![](images/yourimagename.png)

Sizing images

There are a number of ways of sizing images using the knitr package.

1. Size all images.

If we wanted all images to be the same size we could include the following R chunk at the top of the document. This will set the height and width to 6 inches (using the knitr package)

opts_chunk$set(comment=NA, fig.width=6, fig.height=6))

2. Size a single image.

We often want more control over single images. We can handle this with the knitr package by including an R chunk (ctrl alt I on Windows and Cmd I on a Mac).

In the code chunk heading we could specify the width of a figure as percentages as fillows and we might also want to centre the figure {r out.width = "50%", fig.align = "center"}. We would then include a call to knitr::include_graphics() with our image address in quotes as below. Note that the knitr documentation specifies that you may need to install.packages("png") and install.packages("jpeg") for automatic calculation of the width of images.

For most purposes something like this will work for a web page display.

knitr::include_graphics("images/fig31_adjust_nodes.png")
